# Справочник команд — QA Automation Sandbox

## Запуск тестов

### По набору
```bash
# Smoke (~30 сек)
npx playwright test e2e/smoke.spec.ts --project=chromium

# Sanity (~2 мин)
npx playwright test e2e/sanity.spec.ts --project=chromium

# Chromium only (~5 мин)
npx playwright test --project=chromium

# Full suite (~30 мин)
npm test

# Только mobile
npx playwright test --project="Mobile Safari"
npx playwright test --project="Mobile Safari Plus"
npx playwright test --project="Mobile Chrome"
```

### По браузерам
```bash
# Desktop
npx playwright test --project=chromium

# Mobile Safari (iPhone 15 Pro)
npx playwright test --project="Mobile Safari"

# Mobile Safari Plus (iPhone 15 Pro Max)
npx playwright test --project="Mobile Safari Plus"

# Mobile Chrome (Pixel 5)
npx playwright test --project="Mobile Chrome"
```

### Параллельно (Terminal 1, 2, 3...)
```bash
# Terminal 1
npx playwright test --project=chromium

# Terminal 2
npx playwright test --project="Mobile Safari"

# Terminal 3
npx playwright test --project="Mobile Safari Plus"
```

## Docker

```bash
# Запуск
docker-compose up

# Остановка
docker-compose down

# Перезапуск с нуля
docker-compose down -v
docker-compose up -d
```

## Playwright

```bash
# Установка браузеров
npx playwright install webkit
npx playwright install firefox

# Просмотр отчёта
npx playwright show-report

# Режим UI
npx playwright test --ui

# Workers
npx playwright test --workers=8

# Тrace viewer
npx playwright show-trace trace.zip
```
